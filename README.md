# :book: Web_Backend - 백엔드 동작 공부하기
## 1) 개요
> 백엔드 동작 과정과 데이터베이스 쿼리에 대한 내용들을 간단하게 공부합니다.
1. Maven 프로젝트 이용
2. JDBC 이용
3. Servlet 이용
4. MySQL 쿼리문 공부(DDL, DML, DCL)

## 2) 개발 환경
- MySQL
- Maven(Java)
- Spring(Java)
- Apache Tomcat(웹 서버)
